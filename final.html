Final Exam

Q1:

Acquire, process, and understand the data for the exam:

  1. Download the final exam data collection from: http://karlbenedict.com/GEOG485-585/final-exam/final_data.zip.

  2. Unzip the archive.

  3. Run ogrinfo or gdalinfo (as appropriate) for each of the datasets. Include the first five (5) lines of the output from ogrinfo or gdalinfo for each of the datasets in your writeup.

  4. Place the data in your data directory on the class server.

Q2:

Add the data to GeoServer, create stores and layers.

  1. Add each of the layers to GeoServer through the creation of a new “store” (in your workspace) for each of the datasets.

  2. Create one new “layer” for each of the stores. Include a screenshot of each of the five new layers from the GeoServer “Layer Preview” tool in your writeup.

Q3:

Display any three of the GeoServer hosted layers in QGIS or ArcGIS (as a WMS layer) and include screenshots of each of the three layers in your writeup.

Q4:

Create a new OpenLayers web mapping client:

  1. Include each of the five layers that you created in GeoServer in the layer picker in your mapping client.

  2. Embed the client in your final exam web page.

Q5:

Create and add styled layers to your OpenLayers client:

  1. Create two new layers based on two of the stores added in 2) above.

  2. Create two custom styles, one for each of the two layers and make those the default style for the two new layers

  3. Add these two styled layers to the mapping client generated for question 4. Provide the layer names for your customized layers in your writeup.
